# Solucionador Web Interactivo de Métodos Numéricos

## Descripción General

Aplicación web desarrollada en Python para resolver y visualizar paso a paso los siguientes métodos numéricos:

- Método de la Secante
- Método de Gauss-Seidel
- Método de Gauss-Jordan

La aplicación permite ingresar los datos necesarios para cada método y obtener los resultados correspondientes mediante una interfaz web interactiva.

## Requisitos del Sistema

### Software Requerido

- Python 3.12 o superior
- Navegador web actualizado

### Dependencias

- Streamlit
- NumPy
- SymPy
- Pandas
- Plotly

---

## Instalación

### Clonar el repositorio

```bash
git clone https://github.com/usuario/proyecto-metodos-numericos.git
```

### Ingresar al directorio

```bash
cd proyecto-metodos-numericos
```

### Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## Ejecución

### Ejecutar la aplicación

```bash
streamlit run app.py
```

### Acceder desde el navegador

