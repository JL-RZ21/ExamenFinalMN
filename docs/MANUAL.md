# MANUAL DE USUARIO

## Solucionador Numérico Interactivo

### Introducción

Bienvenido al Solucionador Numérico Interactivo. Esta aplicación permite resolver ecuaciones y sistemas de ecuaciones utilizando diferentes métodos numéricos de forma sencilla e intuitiva. El usuario únicamente debe ingresar los datos solicitados y el sistema mostrará automáticamente los resultados obtenidos.

---

## Acceso a la Aplicación

Al abrir la aplicación, se presentarán las opciones disponibles para seleccionar el método numérico que se desea utilizar.

Los métodos disponibles son:

- Método de la Secante
- Método de Gauss-Seidel
- Método de Gauss-Jordan

---

## Método de la Secante

### ¿Para qué sirve?

Este método permite encontrar una aproximación de la raíz de una función matemática.

### ¿Cómo utilizarlo?

1. Seleccione la opción **Método de la Secante**.
2. Ingrese la función matemática que desea resolver.
3. Introduzca los valores iniciales solicitados.
4. Ingrese la tolerancia requerida.
5. Presione el botón **Calcular**.

### Resultado

El sistema mostrará:

- La raíz aproximada de la función.
- El número de iteraciones realizadas.
- Una tabla con los resultados obtenidos.
- La gráfica correspondiente.

## Método de Gauss-Seidel

### ¿Para qué sirve?

Este método permite resolver sistemas de ecuaciones lineales mediante aproximaciones sucesivas.

### ¿Cómo utilizarlo?

1. Seleccione la opción **Gauss-Seidel**.
2. Ingrese los coeficientes del sistema de ecuaciones.
3. Introduzca los valores solicitados.
4. Presione el botón **Calcular**.

### Resultado

El sistema mostrará:

- La solución aproximada del sistema.
- El error obtenido.
- Las iteraciones realizadas durante el proceso.

**Figura 2. Pantalla del Método de Gauss-Seidel**

![Método de Gauss-Seidel](imagenes/gauss_seidel.png)

## Método de Gauss-Jordan

### ¿Para qué sirve?

Este método permite resolver sistemas de ecuaciones lineales mediante operaciones sobre matrices.

### ¿Cómo utilizarlo?

1. Seleccione la opción **Gauss-Jordan**.
2. Ingrese los datos del sistema de ecuaciones.
3. Presione el botón **Calcular**.

### Resultado

El sistema mostrará:

- La solución final del sistema.
- La matriz resultante.
- El procedimiento realizado por el método.

## Recomendaciones de Uso

- Verifique que los datos ingresados sean correctos antes de ejecutar los cálculos.
- Utilice valores iniciales adecuados para obtener mejores resultados.
- Revise los resultados mostrados por el sistema antes de realizar conclusiones.
- Consulte las tablas y gráficas generadas para interpretar mejor la solución obtenida.

## Cierre de la Aplicación

Para finalizar el uso de la aplicación, simplemente cierre la ventana del navegador donde se encuentra abierta.

El Solucionador Numérico Interactivo facilita la resolución de ecuaciones y sistemas de ecuaciones mediante herramientas numéricas de fácil utilización. Gracias a su interfaz intuitiva, los usuarios pueden ingresar datos, ejecutar cálculos y analizar resultados de manera rápida y eficiente.
